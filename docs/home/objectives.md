# Learning Objectives

By the end of this training programme, you will be able to:

## Technical Competencies

### Prompt Engineering

- [x] **Master the AUTOMAT Framework** for structured scientific prompts
- [x] **Apply the CO-STAR Framework** for context-rich communication
- [x] **Design high-fidelity outputs** suitable for rigorous scientific enquiry
- [x] **Engineer constraints** to focus AI on specific scientific domains

### Model Understanding

- [x] **Distinguish between NLMs and LLMs** and select appropriate tools
- [x] **Understand hallucination mechanisms** and implement verification protocols
- [x] **Navigate context windows** for complex scientific tasks
- [x] **Optimise token usage** for efficient inference

## Operational Competencies

### Security & Compliance

- [x] **Identify sensitive data** that must never be shared with public models
- [x] **Apply the Red List protocol** for IP protection
- [x] **Sanitise data** before processing with external tools
- [x] **Implement air-gapped workflows** for confidential research

### Sustainability

- [x] **Measure carbon footprint** of AI workflows
- [x] **Apply Green AI principles** to minimise energy consumption
- [x] **Batch queries efficiently** to reduce computational waste
- [x] **Choose appropriate model sizes** for different tasks

## Practical Applications

### Materials Science Workflows

- [x] **Automate literature synthesis** for polymer chemistry and nanotechnology
- [x] **Format synthesis protocols** in standardised templates
- [x] **Generate data analysis scripts** for tensile testing and SEM imaging
- [x] **Extract structured data** from unstructured research documents

### Quality Assurance

- [x] **Verify AI outputs** against authoritative sources
- [x] **Detect and correct hallucinations** in technical content
- [x] **Audit AI-generated reports** for scientific accuracy
- [x] **Implement human-in-the-loop** workflows

## Ethical Awareness

- [x] **Recognise bias** in AI outputs
- [x] **Understand data rights** and training data provenance
- [x] **Navigate environmental implications** of AI deployment
- [x] **Apply responsible AI principles** in daily workflows

---

## Assessment Criteria

Throughout the workshop, you'll demonstrate these competencies through:

1. **Interactive Exercises**: "The Hallucination Hunt", "Green Optimisation Challenge"
2. **Real-World Applications**: Processing actual tensile test data and SEM images
3. **Prompt Refinement**: Iterative improvement of scientific queries
4. **Peer Review**: Evaluating and improving colleague's prompts
