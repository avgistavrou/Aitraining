# Accessing the Sandbox

During the workshop, you'll practice prompt engineering in a **secure, local AI environment** using Llama 3.2.

!!! success "No Installation Required"
    Everything runs on the instructor's laptop. You only need a web browser!

---

## Connection Details

### Workshop URL

**Connect to:** `http://[INSTRUCTOR_IP]:3000`

Your instructor will provide the specific IP address at the start of the workshop.

---

## First-Time Setup

### Step 1: Access the URL

Open your browser and navigate to the provided URL.

### Step 2: Login

**If you have credentials:** Login with the username and password provided by your instructor.

**If creating account:** Your instructor will guide you through account creation during the session.

---

## Using the Sandbox

### Select the Model

1. Click **"New Chat"** to start a conversation
2. In the model dropdown (top of chat), select: **llama3.2:3b**
3. Start typing your prompt!

### Test Your Connection

Try this simple prompt:

```
What are non-oxide ceramics?
```

You should receive a response within 5-10 seconds.

## Troubleshooting

### Can't Connect to URL

**Check:**

- Are you on the same WiFi network as the instructor?
- Is the URL typed correctly (including `http://`)?
- Try refreshing the page (F5)

**Ask instructor for help** if issues persist.

---

### Model Not Responding

**Try:**

- Wait 10-15 seconds (CPU inference can be slow)
- Check if model is selected correctly (llama3.2:3b)
- Start a new chat

---

### Forgot Login Credentials

**Ask your instructor** to reset your password or provide credentials again.
